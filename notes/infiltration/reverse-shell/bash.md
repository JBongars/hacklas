### Make fully interactive

**Method 1**

```bash
python3 -c 'import pty;pty.spawn("/bin/bash")'
CTRL-Z
stty raw -echo
fg
export TERM=xterm
```

**Method 2**

```bash
# Step 1: Spawn PTY
python3 -c 'import pty; pty.spawn("/bin/bash")'

# Step 2: Background shell
# Press: Ctrl+Z

# Step 3: In your local terminal
stty raw -echo; fg

# Step 4: Back in reverse shell
export TERM=xterm
export SHELL=/bin/bash
stty rows 24 columns 80
```
